# SpringBoard (System-Wide)  

**App User Interface**
- Select dark/light apps  

**Power**
- Disable power vibration
- Place device screen down to lock

**Dock**
- Hide completely
- Background style (transparent, dark, light)
- Use Floating Dock

    - Remove divider
    - Disable App Library icon (iOS 15+)

**Icon Pages**
